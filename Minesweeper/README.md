# 扫雷游戏

一个纯客户端的网页版扫雷游戏，支持在任何设备上运行，包括离线状态。

## 游戏特性

- **三种难度级别**：简单（9x9，10雷）、中等（16x16，40雷）、困难（16x30，99雷）
- **实时计时器**：记录游戏用时
- **地雷计数器**：显示剩余地雷数量
- **标记功能**：
  - 左键点击/触摸：揭示单元格
  - 右键点击/长按：循环标记状态（未标记 → 地雷标记 → 问号标记 → 未标记）
- **朋克科技风格**：炫酷的霓虹效果和深色背景
- **响应式设计**：适配桌面和移动设备

## 如何运行

### 方法1：直接打开HTML文件（推荐）
1. 下载整个游戏文件夹
2. 在电脑或手机上找到 `index.html` 文件
3. 双击或点击打开该文件，游戏会在默认浏览器中运行
4. 无需网络连接，完全离线运行

### 方法2：使用本地服务器（可选）
如果直接打开HTML文件遇到安全限制，可以使用本地服务器：

**Windows**：
```bash
# 使用Python 3
python -m http.server 8000

# 或使用Python 2
python -m SimpleHTTPServer 8000
```

**Mac/Linux**：
```bash
# 使用Python 3
python3 -m http.server 8000

# 或使用Python 2
python -m SimpleHTTPServer 8000
```

然后在浏览器中访问 `http://localhost:8000`

## 技术实现

- **HTML5**：游戏界面结构
- **CSS3**：游戏样式和动画效果
- **JavaScript (ES6)**：游戏逻辑
- **纯客户端**：不依赖任何外部资源或服务器

## 移动设备操作

- **点击**：揭示单元格
- **长按**：循环标记状态（约500ms）

## 游戏规则

1. 点击单元格揭示内容
2. 数字表示周围8个单元格中的地雷数量
3. 标记疑似地雷的位置
4. 揭示所有非地雷单元格获胜
5. 点击到地雷游戏结束

 enjoy playing!
